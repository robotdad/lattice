# amplifier-module-tool-m365-graph

Microsoft Graph API tool module for Amplifier agents.

## Features

- Email operations (read, send, search)
- Calendar management
- File operations via OneDrive/SharePoint
- User profile access

## Installation

```bash
uv pip install amplifier-module-tool-m365-graph
```

## Dependencies

Requires `amplifier-module-tool-m365-auth` for authentication.

## Usage

```python
from amplifier_module_tool_m365_graph import mount

# Mount registers the Graph API tools with the agent
```
